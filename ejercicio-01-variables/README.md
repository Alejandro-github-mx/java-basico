# Ejercicio 01 — Variables y tipos de datos

## ¿Qué hace este programa?
Declara variables de tipo `String`, `int` y `double` para
almacenar datos personales e imprime un mensaje formateado.

## Conceptos practicados
- Declaración de variables con tipos primitivos
- Diferencia entre `String`, `int` y `double`
- Concatenación con `+` en `System.out.println()`

## Lo que aprendí
Entendí que Java es un lenguaje de tipado estático: hay que
declarar el tipo de cada variable antes de usarla.

## Cómo ejecutar
```bash
javac Presentacion.java
java Presentacion
```
