# java-basico
Repositorio de ejercicios para aprender los fundamentos de Java. Cada ejercicio está documentado con el concepto trabajado y el aprendizaje obtenido.

## Objetivo
Dominar los conceptos base de Java como parte de mi formación para desarrollarme como programador/a.

## Temas cubiertos

| No. | Tema | Archivo
| 01 | Variables y tipos de datos | "ejercicio-01-variables/" |
| 02 | Condicionales (if/else) | "ejercicio-02-condicionales/" |
| 03 | Bucles (for) | "ejercicio-03-bucles/" |
| 04 | Métodos y retorno de valores | "ejercicio-04-metodos/" |
| 05 | Arrays | "ejercicio-05-arrays/" |

## Tecnologías

- Java
- Github Codespaces

## Cómo ejecutar un ejercicio
```bash
cd ejercicio-01-variables
javac Presentacion.java
java Presentacion
```
